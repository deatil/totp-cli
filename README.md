# Totp-cli

totp-cli is generate code cli tool.


### Env

 - Zig >= 0.14.0-dev.2851+b074fb7dd


### Download

~~~cmd
git clone github.com/deatil/totp-cli
~~~


### Get Starting

zig build
~~~cmd
zig build install
~~~

use cli and use `conf.json`
~~~cmd
totp-cli
~~~
